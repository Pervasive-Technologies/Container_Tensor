# Purpose

Baked machine learning framework stack, CUDA backend

### Feature

- TensorFlow
- Keras
- PyTorch
- OpenCV
- Jupyter Notebook

### Python packages

- Python 3.6.5
- TensorFlow 1.8.0
- Keras 2.1.6
- OpenCV 3.4.0
- PyTorch 0.4.0
- Numpy
- Matplotlib
- Scipy
- Pandas
- Seaborn
- Scikit-learn
- Scikit-image
- Tornado
- Jupyter notebook
- Intel Math Kernel Library (MKL)
- FFMpeg

### Usage

Docker pull command
```
$ docker pull rainbean/tensor
```
