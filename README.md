# Purpose

Fork from official TensorFlow repository, optimize for development

### Feature

- Python 3.5
- Intel Math Kernel Library (MKL)
- Optimize for CPU parallelism (eg. SSE)

### Python packages

- TensorFlow 1.5.0
- Keras
- Numpy
- Matplotlib
- Scipy
- Pandas
- Seaborn
- Scikit-learn
- Tornado
- Protobuf
- Jupyter notebook

### Usage

Docker pull command
```
$ docker pull rainbean/tensorflow
```
